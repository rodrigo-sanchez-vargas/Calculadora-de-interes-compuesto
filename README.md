# Calculadora de interés compuesto

## Descripción: El código busca principalmente dar a conocer el gran beneficio de utilizar cuentas de ahorro con interés compuesto, ayudando al usuario del código a visualizar el impacto en su dinero que tendrá tener sus ahorros en una cuenta de este tipo.

## Lenguaje: Python.

## Conocimientos utilizados: Manejo de errores, funciones, matemáticas básicas.
